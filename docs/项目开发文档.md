## 七、关键功能实现记录

### 7.1 Feather Icons SVG 图标（v1.8.2）

**问题**：uni-app `<text>` 标签内写 SVG 会被当纯文本渲染。

**解决方案**：将 SVG 字符串定义在 `data()` 中，模板使用 `<view v-html="svgVar">` 渲染。

### 7.2 播放器核心功能

- **横竖屏**：依赖 `onFullscreenChange` 事件，不在 `toggleFullscreen()` 中直接改状态
- **续播**：存 `play_history_{vodId}` 记录（position, duration, epIndex）
- **全局观看历史**（mine 页面）：需在 `saveProgress()` 中同时更新 `play_history` 数组
- **选集历史标记**：`isEpisodeWatched()` 读取 `play_history` 全局数组
- **跳过片头片尾**：根据服务端集数数据（每集时间戳）判断

### 7.3 分类页并发加载

`IntersectionObserver` 在 `scroll-view` 内不触发，改用 `Promise.all` 并发加载所有子分类，合并去重后前端筛选。

### 7.4 发现页重构

使用原生 `getVodList` + 并发子分类合并 + 前端筛选，不再依赖懒加载。

## 八、常见问题

### Q1: APK 安装失败
- 检查签名是否正确（必须用 test.jks）
- 检查 APK 大小（正常 ~23 MB，太小说明 lib/ 和 classes2.dex 丢失）
- 禁止用 Python repack + apksigner 重签名

### Q2: appkey 配置错误
- 说明签名指纹变了，必须用 Gradle + test.jks 重新构建
- 禁止手动重签名 APK

### Q3: Feather 图标显示为纯文本
- 确保 SVG 放在 `<view v-html>` 中，不是 `<text>` 中
- SVG 字符串定义在 `data()` 里

### Q4: 观看历史不显示
- 确认 `player/index.vue` 的 `saveProgress()` 中有更新全局 `play_history` 数组
- mine 页面读取的是全局数组，不是 `play_history_{vodId}`
