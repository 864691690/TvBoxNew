# TvBoxNew 项目创建 - 2026-06-02 09:39

## Objective
按照用户提供的「全新真实可落地｜类极光TV影视大屏APP开发方案（苹果CMS V10对接）」方案，创建 uni-app TV 项目骨架。

## Key Decisions
1. **技术栈**: uni-app + Vue3 (模板默认, Options API 兼容 Vue2 语法) + Vite
2. **数据源**: AppleCMS V10 @ `https://move.wuyaoxuexi.cn`
3. **先用 Vue3 模板**: Options API 代码在 Vue3 中完全兼容，后续可降级

## Files Created
- `src/pages.json` - 6个页面路由 + tabBar配置
- `src/manifest.json` - 应用配置 (横屏、TV focus)
- `src/App.vue` - 全局样式
- `src/api/config.js` - API基础URL + 分类常量
- `src/api/vod.js` - 5个苹果CMS接口封装
- `src/utils/parser.js` - 播放源解析 (集数$链接 格式)
- `src/utils/cache.js` - 本地缓存工具 (2小时默认)
- `src/utils/focus.js` - TV焦点管理工具
- `src/pages/index/index.vue` - 首页 (轮播、热门、最新、分类列表)
- `src/pages/category/index.vue` - 分类页 (筛选标签 + 宫格)
- `src/pages/detail/index.vue` - 详情页 (选集、多线路、收藏)
- `src/pages/player/index.vue` - 播放页 (video组件、选集切换、进度记忆)
- `src/pages/search/index.vue` - 搜索页 (热搜、历史、结果)
- `src/pages/mine/index.vue` - 个人中心 (历史、收藏、设置)
